Prototype hint system page

Student version of Introductory Combinatorics by Guided Discovery with hyperlinks for all cross references and online hints for many problems.

This link will download a pdf file of the notes for Math 28 that has links to hints. You will need to use Adobe Acrobat Reader to see the links in this file.  (You can download the Adobe Reader free of charge at www.Adobe.com.) When you click on a link to a hint, it will download from the web a pdf file containing a hint. You can use the back arrow at the bottom of the Adobe Reader window to return to where you were before you clicked on the link to the hint.

.

From time to time, the system will ask you for your DND user name and password. Your DND name is the name you find when you look yourself up with "Look up a Name" under the BlitzMail menu. Once you have downloaded this file, you can simply use the file to look up hints; you do not need to return to this page to get to the hints.

 

This pdf file also has hyperlinks; if you click on a highlighted cross reference, index entry, or table of contents entry, then you are taken to the referenced part of the file.  You can use the back arrow at the bottom of the Adobe Reader window to return to where you were before you clicked on the highlighted entry.
